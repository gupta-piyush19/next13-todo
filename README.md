# Todo App

## Description

This is a simple todo app that allows you to add, edit, and delete tasks. It also allows you to mark
tasks as complete.

## Installation

1. Clone the repository
2. Run `yarn`
3. Run `yarn run dev`
4. Open `http://localhost:3000` in your browser
5. Enjoy!

## Technologies Used

- Next13(Server components + app directory)
- TailwindCSS
- Prisma
- sqlite
- eslint
- prettier
- husky
- lint-staged
